# RuSafetyBench

Open adversarial benchmark for Russian-language LLM safety evaluation.

Early-stage structured red-teaming dataset focused on Russian-language model behavior across social engineering, gradual escalation, geopolitical misinformation, historical revisionism, and creative fiction bypass.

Built as part of the [LLM Safety Evaluation Lab](https://github.com/fsrxc2bvv9-ctrl/llm-safety-evaluation-lab) project.

## Models Evaluated

ChatGPT, GigaChat, Alisa, Meta AI, DeepSeek
